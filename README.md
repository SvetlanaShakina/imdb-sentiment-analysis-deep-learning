# IMDB Sentiment Analysis Using Deep Learning

## Project Description

This project performs sentiment analysis on movie reviews from the IMDB dataset.

The objective is to classify reviews as positive or negative using a deep learning model.

## Dataset

IMDB Dataset of 50K Movie Reviews

Source:
https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews

## Technologies

- Python
- TensorFlow / Keras
- Pandas
- NumPy
- Matplotlib
- NLTK

## Model Architecture

- Embedding Layer
- GlobalAveragePooling1D
- Dense Layer (ReLU)
- Dropout
- Output Layer (Sigmoid)

## Results

Test Accuracy: 86.67%

The model successfully classifies positive and negative movie reviews.
